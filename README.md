## SQLite

### FOREIGN KEY

> В SQLite3 по умолчанию отключена поддержка внешних ключей, команда PRAGMA позволяет включить внешние ключи в базах данных SQLite

Включение поддержки FOREIGN KEY:

```sql
PRAGMA foreign_keys=on;
```
